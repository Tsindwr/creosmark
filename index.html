🎮 Get Started

  • Character Builder: Create and customize your character's stats. Roll dice or manually enter values for the 8 Sunder potentials
  • Character Sheet: View and interact with your character's potentials using the visual potential tracks
  • Your character data is automatically saved in your browser

📖 Potential Track Usage

  • Center circle: Left-click (or tap) to add stress, right-click (or long-press) to add resistance
  • Nodes: Click to increment/decrement by 1 on that side
  • Stress (red) fills from the left - represents damage, exhaustion, or strain
  • Resistance (green) fills from the right - represents armor, protection, or buffs
  • The center number shows your total potential score
  • Total stress + resistance can never exceed your score
  • Keyboard accessible: Tab to navigate, Enter/Space for stress, R for resistance

💻 Developer Usage

<!-- Include the component -->
<script type="module" src="src/components/potential-track.js"></script>
<link rel="stylesheet" href="src/css/potential-track.css">

<!-- Use the component -->
<potential-track label="Body" score="10" stress="0" resistance="0"></potential-track>

<!-- JavaScript API -->
<script>
  const track = document.querySelector('potential-track');
  track.addStress();        // Add 1 stress
  track.removeStress();     // Remove 1 stress
  track.addResistance();    // Add 1 resistance
  track.removeResistance(); // Remove 1 resistance
  
  // Listen for changes
  track.addEventListener('stress-changed', (e) => {
    console.log(e.detail); // { stress, resistance, score }
  });
</script>
