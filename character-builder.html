📖 How to Use

  • Manual Entry: Type values directly into the input fields (1-18)
  • Roll Dice: Click the "Roll" button to generate a random stat value
  • Roll Formula: 3d6 drop the highest die (special case: if all three dice are 1, the result is 1)
  • View Sheet: Click "View Character Sheet" to see your stats displayed with potential tracks
  • Your stats are automatically saved as you make changes
