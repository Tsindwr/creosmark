📖 How to Use

  • Roll Stats: Click "Roll 8 Stats" to generate 8 random values using 3d6 drop highest
  • Assign Stats (Drag & Drop): Drag rolled values from the pool and drop them onto your desired potentials
  • Assign Stats (Click): Click a stat value in the pool to select it (gold border), then click a potential to assign it
  • Manual Entry: You can also type values directly into the input fields (1-18)
  • Clear Assignment: Click "Clear" button to return a stat value back to the pool
  • View Sheet: Click "View Character Sheet" to see your stats displayed with potential tracks
  • Your stats are automatically saved as you make changes
